## 📖 Présentation

Student Manager est une application web développée avec **Laravel** permettant de gérer facilement les étudiants et leurs filières.

Ce projet a été réalisé dans le cadre de ma formation en **Génie Logiciel** afin de mettre en pratique le développement d'une application CRUD complète avec Laravel.

## 🚀 Installation

### 1. Cloner le projet

```bash
git clone https://github.com/Fatou03/student-manager.git
```

### 2. Entrer dans le dossier

```bash
cd student-manager
```

### 3. Installer les dépendances

```bash
composer install
```

### 4. Copier le fichier d'environnement

```bash
cp .env.example .env
```

### 5. Générer la clé Laravel

```bash
php artisan key:generate
```

### 6. Configurer la base de données

Modifier les informations de connexion dans le fichier `.env`.

### 7. Exécuter les migrations

```bash
php artisan migrate
```

### 8. Lancer le serveur

```bash
php artisan serve
```
